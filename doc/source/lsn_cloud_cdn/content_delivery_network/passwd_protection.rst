What is Password Protection?
============================

The Password Protection feature of LSN CDN allows you to only allow certain
people access to your content. These people must login with a username and
password.

**How to Enable Protection**

1. Login to OnePortal’s cdn.html" target="_blank">CDN setup page.
2. Click “Modify Password Policy”.
3. Click “Enabled”.
4. Click “Add new user”, enter a username and password for the new user and
   click “Save Changes”.
5. Repeat the previous step for any users you would like to add.
6. click “Save Changes” when finished.
